# tech-blog

`tech-blog` is a Jekyll Chirpy site for `tech.henjini.com`.

The site publishes Korean practical technical posts for developers who want clear working notes around frontend development, TypeScript, Node.js, databases, Linux, Docker, DevOps, testing, security, observability, and production troubleshooting.

## Content Rules

- Prefer practical explanations that connect concepts to real development or operation work.
- Include runnable code, commands, configuration, or checklists when the topic naturally needs them.
- Treat security, database migration, infrastructure, and cost-related advice as verification-sensitive.
- Use concrete titles that match developer search intent and describe the problem or technique directly.
- Keep filenames and slugs in lowercase English kebab-case.

## Local Notes

The site uses the Chirpy theme. Install dependencies before building locally.
